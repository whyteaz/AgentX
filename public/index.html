How It Works

Enter a tweet link below and choose whether to enable "Troll Lord" mode. In standard mode, our AI agent fetches the tweet, generates a spicy trolling response using Google Gemini, replies on Twitter, and logs the action on NEAR.

Troll Lord Mode: When enabled, the agent automatically replies to the target tweet 10 times at 16-minute intervals. The UI switches to dark mode (including input text), and an image appears next to the toggle. The response section updates with each reply's status until all 10 replies are complete.

Note: In standard mode, you can only trigger the AI agent once every 15 minutes.
